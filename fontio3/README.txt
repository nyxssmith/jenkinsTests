fontio3
=======

fontio3 is a comprehensive library for working with sfnt-housed fonts. It includes support for modern OpenType tables including variable fonts (OpenType 1.8 and later) and much more.

To get started, just install with `$ python setup.py install` and you will be able to use it in your Python projects.
